## :information_source: Run the project

```bash
# Clone the repo
$ git clone https://github.com/Svaity/Netflix-Clone.git

# Get into the directory
$ cd Netflix-Clone-master

#Install the dependancies
$ npm  install

#Run the server
$ npm start

```
